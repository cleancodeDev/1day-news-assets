# Privacy Policy — cardApp (1DAY 뉴스피드)

**Last updated: 2026-07-22**

cardApp is a personal content-publishing tool used solely to publish news
curation cards to the Instagram account **@1day.news.feed**, which is owned
and operated by the developer of this app.

## Data Collection
This app does **not** collect, store, or share any personal data from any
user. It does not have end users other than the developer. It accesses only
the developer's own Instagram professional account via the official
Instagram API to upload content created by the developer.

## Data Usage
The only data handled by this app is:
- Content (images, videos, captions) created by the developer for publishing
- The developer's own Instagram account ID and access token

No third-party data is processed. No cookies, analytics, or tracking are used.

## Data Deletion
This app stores no user data, so there is nothing to delete. To revoke this
app's access to the Instagram account, remove it at
Instagram → Settings → Apps and Websites.
